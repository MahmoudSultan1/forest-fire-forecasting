To Run:
1-Navigate to build directory
2-Open terminal and type the command "cmake .." (make file will be generated)
3-type the command "make" ( a three executables will be generated "server, client, test)
4-Run the server first using "./server" (the server will wait for a client to connect at the same port)
5-Run the client usning "./client" 

for testing Run the test using ./test
